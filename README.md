Interactive Portfolio

An interactive, desktop-first developer portfolio showcasing my work, skills, and journey as a full-stack developer. Built with a strong focus on clean UI, structured layouts, smooth animations, and real-world integrations.

FEATURES
• Custom grid-based editorial layout
• Interactive hero section with animated software icons
• Backend-driven projects showcase
• GSAP-powered scroll timeline
• AI chatbot using OpenRouter API
• Embedded “Why You Should Hire Me” video
• Client-side routing with React Router
• Desktop-first UI with scaled mobile support

TECH STACK
Frontend:
React, Tailwind CSS, GSAP, React Router DOM

Backend:
Node.js, Express.js, MongoDB

AI:
OpenRouter API
